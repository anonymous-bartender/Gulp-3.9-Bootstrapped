# Vanilla Gulp

Boilarplate for Front End developers with Gulp 3.9.1, FontAwesome 4, Bootstrap 4, JQuery, HTML5, SCSS

## Instructions
```bash
git clone https://github.com/arghajit/vanilla_gulp.git
npm run dev

# or through `gulp`
gulp serve
```
